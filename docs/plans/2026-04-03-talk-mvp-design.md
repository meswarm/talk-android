# Talk — Matrix 聊天客户端 MVP 设计文档

## 概述

开发一款基于 Matrix 协议的 Android 聊天应用，用于与 AI Agent 进行对话交互。
界面参考微信风格，主打简洁高效，支持高质量 Markdown 渲染和明暗双主题。

## 环境信息

- **Matrix Homeserver**: Conduit，本地或私有网络部署
- **用户账号**: 使用本地测试账号，密码不写入文档或仓库
- **Agent 账号**: 使用本地测试 Agent 账号
- **开发设备**: Android 手机通过 USB 连接，同一局域网

## MVP 范围（第一阶段）

### 包含
- 登录页面（服务器地址 + 用户名密码）
- 会话列表页（房间列表、最新消息、未读标记）
- 聊天页（气泡样式、Markdown 渲染、流式消息接收）
- 文本消息发送
- 本地聊天记录持久化（matrix SDK 内置 Hive）
- 明暗双主题，跟随系统自动切换

### 不包含（后续阶段）
- 富文本 Markdown 编辑器（输入框）
- 图片/视频/文件插入与上传
- 个人资料编辑
- 系统通知推送
- 消息发送状态（发送中/成功/失败）

## 技术选型

| 技术 | 选择 | 理由 |
|------|------|------|
| 框架 | Flutter | 已有项目基础 |
| Matrix SDK | `matrix` dart 包 | 社区最成熟，FluffyChat 同款 |
| Markdown | `markdown_widget` | 高度可定制，支持 LaTeX/代码高亮 |
| 状态管理 | `provider` | 轻量，适合 MVP |
| 本地存储 | matrix SDK 内置 Hive | 无需额外配置 |

## 架构

```
UI 层 (Pages/Widgets)
    ↓
状态管理层 (Provider)
    ↓
服务层 (MatrixService)
    ↓
matrix dart SDK
```

## 页面设计

### 1. 登录页
- 服务器地址输入（预填）
- 用户名 + 密码
- 登录按钮 + 加载状态
- 自动登录记忆

### 2. 会话列表页
- 顶栏：应用名 + 主题切换
- 列表：头像、昵称、最新消息、时间、未读红点
- 点击进入聊天

### 3. 聊天页
- 消息气泡（我方右侧绿色，对方左侧白/灰色）
- markdown_widget 渲染消息内容
- 流式消息实时更新
- 底部文本输入框 + 发送按钮
